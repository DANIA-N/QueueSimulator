# QueueSimulator Maintenance – SWE 441

This repository contains the maintenance work completed for the QueueSimulator project as part of the Software Maintenance and Evolution (SWE 441) course.

The project focuses on applying software maintenance practices using Jira Software, GitHub, and Confluence.

---

## 📌 Project Overview

The goal of this project was to maintain and enhance the QueueSimulator system by:

- Fixing bugs
- Adding new features
- Improving code readability and output formatting
- Reducing cyclomatic complexity
- Enhancing documentation
- Tracking work using Jira
- Documenting progress using Confluence

Eight maintenance issues (KAN-5 to KAN-12) were completed.

---

## 🛠 Completed Maintenance Issues

Each issue was linked to code changes and tracked through the custom Jira workflow.

Completed issues include:

- Bug fixes
- Feature requests
- Improvements
- Refactoring
- Documentation updates

All issues were resolved and marked as **Done**.
